## 2. 「決定力」への補足(裁定統合)

THEORY_SYNTHESIS §2 は W の「4役」(慣性フレームの重み・重力ポテンシャル・時間率・屈折率)を
定義済みである。本書は役割の列挙に対し、**役割を貫く解釈**を3点補足する。

- **補足1(慣性のアンカー)〔Gemini、裁定 #8〕** — ある点の決定力 W が大きいとは、
  その点の「静止」「等速直線運動」の基準(局所フレーム u)が、決定力の源である質点の運動に
  強く支配される、ということである。決定力は引力の強さである前に、**慣性の基準の固さ**である。
  重力(E4)・引きずり(E6′)・時間の遅れ(E7R)・光の屈折(E8R)は、この一つのアンカーの
  強度分布から創発する。

- **補足2(二重性と自己完結ループ)〔Grok、裁定 #9〕** — 決定力は
  (a) **空間座標そのものの定義子**(A2: 座標は質点の相対配置からのみ定まり、その「定まりやすさ」が W)、
  (b) **その空間内での慣性の決定子**(A4/A5: u の重み)、という二重の役割を持つ。
  質点配置 → W・u の決定 → 質点の運動更新 → 新しい配置、という**自己完結フィードバックループ**が
  DFM の関係論的力学の核心であり、外部の入れ物(絶対空間)をどこにも必要としない。

- **補足3(持ち分としての決定力)〔本裁定の追加〕** — 点 r における質点 j の寄与率
  φ_j(r) = w_j(r)/W(r)(Σφ + φ_bg = 1)は、「その場所の空間を誰がどれだけ所有して
  いるか」の**持ち分(シェア)**と読める。E6′ の反作用分配(φ_ij 比例)、E7R の自己除外
  (自分の持ち分は自分の時計に入れない)、そして §3 の箱宇宙(持ち分が一つの剛体に
  集中する極限)は、すべてこのシェア解釈の系である。
  平易な一文(EXPLAINER 展開用): **「決定力とは、その場所で『どちらが止まっているか』を
  決める投票権であり、質量が大きく距離が近いほど票数が多い」**。

## 3. 思考実験「大質量の箱」の定式化(定理候補 T10)

### 3.1 設定

宇宙に質量 M_box の**箱**(2D 実装では質点を等間隔に並べた剛体リングまたは矩形殻)だけが
存在し、その内部に微小質量のテスト質点群を置く。D₀ = 0(箱が唯一の背景)、k_frame = 1。
箱要素の総決定力を W_box(r) と書く。内部では w_test ≪ W_box ゆえ φ_box ≈ 1
(空間の持ち分はほぼ完全に箱にある)。

原仮定者の提示4項目は、既存公理の帰結として次の (i)〜(iv) に定式化される。

### 3.2 T10(箱宇宙)— 主張と導出スケッチ

- **(i) 相対座標の具体化** — A2 により空間座標は質点の相対配置からのみ定まる。箱以外に
  基準となる質量がないため、**内部の空間座標は箱に対する相対で完全に決定される**
  (箱の中心・主軸が座標系の原点・軸になる)。
- **(ii) 並進の不可知性(厳密)** — 箱が一様速度 v_box で並進するとき、E3 の分子は全ソースが
  同一速度なので u(r) = v_box が**厳密に**成り立つ。A5 により内部質点は u に対する速度を
  保つため、箱ごと運ばれる。内部で閉じた実験(相対距離・相対速度・時計比較)の可観測量は
  すべて v_box に依存しない — **箱の並進は原理的に検知できない**(T1 の多体拡張)。
- **(iii) 回転の不可知性(中心対称配置で成立、一般には近似)** — 箱が角速度 ω_box で
  剛体回転するとき、各壁要素の速度は ω_box ẑ×(r_j−r_c)。E3 の加重平均は、リング(中心対称)
  配置では対称性により回転場 u(r) = ω_box ẑ×(r−r_c) + O(非対称項) を与え、さらに壁要素の
  スピン引きずり(E2、s_j = ω_box)が壁近傍の剛体接続を補強する。k_frame = 1 で内部質点は
  共回転フレームに乗るため遠心力・コリオリ力を観測せず、**箱の回転も内部から検知できない**。
  ニュートンのバケツの解: 「水面が平らになるのは、水が箱(=宇宙)と共回転しているとき」。
  厳密にどこまで u が剛体場に一致するかは配置依存であり、**V23(数値検証)で機械測定する**
  (誠実性のため: 並進は厳密、回転は対称配置+E2 補強つきの近似、と主張水準を分ける)。
- **(iv) 伸縮の共動(→§4)** — 箱が相似に伸縮すると、壁要素の速度 (L̇/L)(r_j−r_c) が
  E3 を通じて内部に半径方向の速度場を与え、E6′ が内部質点をそれに乗せる。
  **箱の中の質点同士の位置関係は箱の伸縮に比例して伸び縮みする**。

### 3.3 D₀ との関係 — 「背景」の実体化

A3 の D₀ は「遠方大規模構造の一様等方な粗視化」だった。箱宇宙は、この無限遠の背景を
**有限距離の明示的なソース群に引き戻した**ものである:

- D₀(静的スカラー・位置も速度も持たない)→ 箱(位置・速度・スピン・サイズ L(t) を持つ動的ソース)
- 「D₀ との運動量交換 = 無限慣性リザーバへの移譲」(A13)→ 箱要素への通常の反作用(帳簿不要の閉鎖系)
- T1(単独星の運動は検出不能)〜 T2(引きずり)〜 箱宇宙(背景ごと運ばれる)が一本の系列になる

この読み替えにより、**L14(宇宙論的 D₀ の7桁問題)は「どの半径までの物質が『箱』を構成する
とみなせるか(因果地平・遮蔽・膨張による希釈)」という物理の問いに変換される**。
これが箱宇宙の理論的な主収穫である(解決そのものは Phase 3 の課題として残る)。

## 4. 「宇宙の膨張」の内包案(採用案: 膨張する箱)

### 4.1 第1段階 — 運動学(新公理不要・既存式から創発)

膨張則 L(t)(箱の線形サイズ)を所与の関数として壁要素を r_j(t) = (L(t)/L₀)·r_j(0) で動かす。
このとき壁要素の速度は v_j = H(t)·(r_j−r_c)、H(t) ≡ L̇/L。

- **E3**: 内部の局所フレームは u(r) ≈ H(t)·(r−r_c) — **Hubble 型の速度場が加重平均から創発**する。
- **E6′**(k_frame=1): 内部質点は Δv = k_frame·Δu で u の変化に運ばれ、**共動系(comoving)**になる。
  自由質点2つの距離は L(t) に比例して伸びる — 原仮定者の「箱が伸び縮みすると、箱の中の
  質点同士の位置関係も伸び縮みする」がそのまま機械検証可能な予言になる(V24)。
- **束縛系は膨張しない**: 局所の質点対(連星・「銀河」)の近傍では自身の w が箱の持ち分を
  上回り(φ_local > φ_box)、u は局所質点に支配される。ゆえに**強く束縛された系の内部距離は
  L(t) に追従しない** — 「膨張は銀河間には効くが銀河内には効かない」という実宇宙の描像の
  類似がパラメータなしで出る(V25)。
- **赤方偏移の類似(将来)**: E8R の媒質随伴(光は u に運ばれる)の下で、膨張する u 場を
  横断する光の到着波長の伸びを 2D レイトレーサで測定できる(第2段階の観測量)。

Grok 案の「位置更新則に強制項 (L̇/L)r·Δt を追加」は、**E3+E6′ が既にその仕事をする**ため
不要と裁定した(裁定 #12)。強制項方式は A13(相互性)の反作用経路を持たず保存則を壊す。

### 4.2 第2段階 — 動力学(L(t) の力学変数化)

壁要素を pinned にせず、E4(自己重力)・E5′(スピン斥力=圧力)に従わせると、
L(t) は与えるものではなく**解かれるもの**になる:

- 自己重力のみ → 減速膨張(重力的束縛の 2D 類似)
- 壁要素に高スピン(高温)→ E5′ 圧力による膨張駆動
- これは「フリードマン方程式の DFM 類似」(Gemini 拡張案B の実装形)であり、
  D₀(t) の時間発展則を箱の運動方程式として具体化する。

主張水準(Negative claims 継承): これは FLRW 宇宙論の再現主張ではなく、
「関係論的な膨張(距離基準そのものの時間発展)がどの公理から創発するか」の機構デモである。
