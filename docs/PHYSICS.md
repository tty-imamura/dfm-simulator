## 2. 公理系 (A1〜A10)

記号: 質点 i は質量 m_i、位置 **r**_i、速度 **v**_i、スピン s_i(符号付きスカラー、z軸まわり角速度)を持つ。
2次元平面でシミュレートする。半径 R_i = r_R·√m_i(2次元円盤)、慣性モーメント I_i = ½·m_i·R_i²。
d_j = |**r** − **r**_j|、ε はソフトニング長(発散防止)。

- **A1(質点)** — 宇宙は有限個の質点の集合である。空間そのものは実体を持たない。
- **A2(相対空間)** — 空間座標は質点の相対配置からのみ定義される。絶対空間・絶対静止系は存在しない。
- **A3(決定力)** — 質点 j が点 **r** の空間座標に及ぼす「決定力」は
  w_j = m_j/√(d_j²+ε²)〔v1.2: Plummer 型に統一。ε→0 で原仮定 m/d に一致し、
  A6(E4 = G∇W)が有限 ε でも厳密に成立する(L5 解消)〕。
  遠方の大規模宇宙構造は一様等方な**背景決定力 D₀** として寄与する(補完①)。
  総決定力 W(**r**) = D₀ + Σ_j w_j(**r**)。
- **A4(局所フレーム)** — 各点の「空間の速度」**u**(**r**) は、決定力を重みとする各質点の運動
  (並進+スピン回転)の加重平均である。背景 D₀ の寄与は静止(遠方宇宙は平均して動かない)。
- **A5(慣性)** — 質点は**局所フレームに対する**速度 (**v**−**u**) を保とうとする。
  フレーム自体が動けば質点はそれに運ばれる(=引きずり)。
- **A6(重力)** — 決定力の空間勾配が加速度を生む: **a** = G·∇D(**r**)。
  w ∝ m/d なので ∇w ∝ m/d² となり、**ニュートンの逆二乗則が自動的に再現される**。
- **A7′(時間)〔v1.4 改訂・原仮定者承認 2026-07-16〕** — 局所の固有時間の進みは、
  決定力が高いほど、また**共動フレームに対する運動が速いほど**遅い:
  dτ/dt = √( N² − A²·|**v**−**u**|²/c₀² )、 N = e^{−ψ}、 A = e^{+ψ}、 ψ = W_ext/K_t。
  弱場・低速では dτ/dt ≈ 1 − ψ − |**v**−**u**|²/(2c₀²) となり、旧 A7(1/(1+ψ))の
  一次形に運動項が加わった形になる。フレーム不在(W_ext=0)では運動は定義されず
  運動時間遅れも生じない(T1 と整合)。根号内 ≤ 0(局所光速超過)は非物理域。
- **A8(スピン引きずり)** — スピン s_j は周囲のフレームに回転成分 ω_j(d) を与える
  (レンズ=サーリング効果に相当。表面 d→0 で剛体回転 s_j に接続し、遠方で減衰)。
- **A9(熱=スピン)** — スピンの2乗がその質点の熱。温度は等分配則 ½I_i s_i² = ½k_B T_i より
  **T_i ≡ I_i s_i²(k_B = 1 のシミュレータ単位系。= ½ m_i R_i² s_i²)**〔v1.1: 旧略記
  T ∝ s²R² は質量因子を欠くため本定義で置換〕。
  接触時、並進運動の散逸分は接触摩擦を通じてスピンに変換される。
  また近接した質点同士のスピンは拡散し、十分な時間で平衡に達する。
- **A10(スピン斥力)** — スピンが回転させるフレームの中に居る物体は遠心加速度 ω²·d を受け、
  外向きに押される。ゆえに高温(高スピン)の質点は互いに押しのけ合う(=圧力の起源)。
- **A11(光の伝播と同時性)〔v1.1・原仮定者改訂〕** — 光は各点の局所決定フレーム **u**(**r**) に
  対して等方的に速さ c₀/n_eff(**r**) で伝播する(=光は媒質 **u** に運ばれる。E8R)。
  座標時間 t は宇宙で一意(絶対的同時性)であり、固有時間 τ の遅れ(E7)は時計の**動的な**
  遅れであって運動学的な相対性ではない。DFM は優先フレームを持つ理論であることを明示する。
  **〔A11 追補・v1.3〕** この優先フレームは A2 が否定する「物質から独立した絶対空間」では
  なく、A3/A4 により**物質分布そのものから創発する共動フレーム**である(u は決定力加重の
  物質運動平均、D₀ は遠方物質の粗視化。質点を全て除けばフレームも消える=T1)。
  ゆえに A2 と A11 は矛盾しない — 優先フレームは空間の性質ではなく物質配置の性質である
  (DERIVATIONS §10.3)。
- **A12(放射)〔v1.1・設計者補完、2026-07-15 原仮定者承認済み〕** — 温度 T_i を持つ質点は
  光量子を放射する。スペクトルのピーク波長はウィーン型則 λ_peak = b_m/T_i(b_m はモデル定数)、
  光度は Λ_i = η_rad·T_i^p(既定 p=4、シュテファン=ボルツマン型)。E9 の法線減衰 γ_n で
  失われる並進エネルギーは衝突時の閃光として放射される。これによりエネルギー収支が閉じる:
  並進 → スピン(摩擦)or 放射(法線減衰)、スピン → 放射(冷却)。
  **〔A12′・v1.2 拡張〕** 接線摩擦のすべり散逸と、スピン同期(E10′)で失われる回転エネルギーも
  光として放射され radE に計上される。これにより散逸4チャネル(γ_n・接線摩擦・E10′同期・
  E11冷却)が全て帳簿づけされ、第一法則が閉じる(V10 で機械検証)。
  **〔v1.6 主張域〕** 閉性が検証済みの範囲は kRep=0(V10)または固定スピン+kRep>0
  (V10b)であり、スピン変化中の ΔU_rep は帳簿外(L17)。
- **A13(相互性)〔v1.1・原仮定者改訂 ★最重要〕** — すべての相互作用(重力・スピン斥力・
  フレーム引きずり・接触・スピン拡散)は対ごとに作用反作用対称であり、系全体の線運動量と
  全角運動量 L_tot = Σ m_i(**r**_i×**v**_i) + Σ I_i s_i を厳密に保存する。
  背景決定力 D₀ との運動量交換は、静止した無限慣性リザーバ(遠方宇宙)への移譲として
  帳簿づけされる — リンゴに引かれた地球が、感知できないほど僅かに動くのと同じである。

## 3. 帰結(「つまり」の定理化)

- **T1(単独の星の運動は検出不能)** — N=1 かつ D₀=0 のとき、**u**(**r**) ≡ **v**₁ となり、
  慣性の基準が自分自身になる。並進も(スピン引きずりにより)回転も原理的に判別できない。✓法則3
- **T2(空間の引きずり)** — 大質量 j の近傍では w_j が支配的になり **u** ≈ **v**_j。
  空間ごと星に引きずられる。D₀ ≫ Σw なら **u** ≈ 0 で空間は安定。✓法則4,6
- **T3(回転方向の整列)〔v1.2: 2D の帰結として表記修正。回転面整列(3D)は拡張 L7 の帰結〕**
  — 接触摩擦(A9)は非整列な相対運動を選択的に散逸させるため、
  軌道は円形化し、スピンの向きが揃う。→「土星の環」「原始円盤」デモ
- **T4(平坦な回転曲線)** — 円盤銀河では星々の集団回転が **u** に回転成分を与える。
  外縁の星は「回転する空間」に乗るため、必要な向心力が減り、
  ケプラー則より速い回転を維持できる。**ダークマターなしで回転曲線が平坦化**する。→「銀河」デモ
- **T5(重力レンズ)〔v1.4 改訂〕** — A7′ より光の実効速度は c₀/n_eff、
  n_eff(**r**) = A/N = e^{2ψ} ≈ 1 + 2W/K_t(v1.7: k_lens を廃止し時計と同一の ψ から生成)。
  フェルマーの原理により光線は決定力の高い側(質量側)へ曲がる。強い場(ψ ≳ 0.3)では
  光線が捕まって周回する(光子球の光学類似)。→「重力レンズ」デモ
- **T6(圧力と熱平衡)** — A9+A10 により、衝突で生じたスピン(熱)は斥力(圧力)を生み、
  スピン拡散で温度が均一化する(熱力学第0法則に相当)。→「気体」デモ
  **〔v1.6 条件明記〕** E10′ が直接均一化するのは s であり、温度 T=Is² の均一化と
  一致するのは**同一慣性モーメント(単一粒径)の系に限る**(異粒径系では一般に不一致。L9)。
- **T7(保存則)〔v1.1〕** — E4(中心力)+E5′(中心力)+E6′(反作用つき輸送)+E9(接触対力)
  +E10′(I 重み拡散)の下で、D₀=0 なら線運動量と全角運動量(軌道+スピン)は厳密に保存する。
  D₀>0 では保存量の欠損は全てリザーバ勘定と一致する。✓A13
- **T8(非対称レンズ)〔v1.1、v1.4: E8R 対応〕** — A11+E8R の下、スピン s_c の中心天体を挟む対称な平行光線は
  非対称に湾曲し、湾曲差は k_F・u_φ に比例する(Kerr 型フレームドラッグの光学類似)。
- **T9(時間パラドックスの座標化)〔v1.3〕** — 再会する二時計の経過時間差は、各世界線に
  沿う E7 の積分として一意に決まる。観測者ごとの「相互的時間遅れ」と折返し時の遠方時刻割当の
  不連続は、観測者中心の同期座標が生む**表示効果**であり、天動説の座標に現れる惑星の逆行と
  同型である(時計の物理的な逆行・矛盾はどの表示でも生じない)。
  限定: T9 は相対性理論の反証ではない — 標準相対論の内部でも双子問題は固有時間積分で
  一意に解ける。T9 の主張は「DFM の基礎表示では同時性の相対性という表示装置が最初から
  不要になる」という表現の単純化である(対応表・導出・限定の全文は DERIVATIONS §10.2)。

## 4. 調整と補完(原仮定に対して設計者が追加・確定した点)

1. **背景決定力 D₀ の導入** — 「大規模構造に囲まれると安定」を定量化するパラメータ。
   これが無いと u の分母が 0 になり得る(質点が1個も無い点でフレームが未定義)。
   D₀→∞ でニュートンの絶対空間に、D₀→0 で完全な相対空間(マッハ原理)に漸近する。
2. **決定力の距離則の確定〔v1.2 改訂〕** — w = m/√(d²+ε²)(Plummer 型)。
   ∇w = −m·(**r**−**r**_j)/(d²+ε²)^{3/2} となり **E4 = G·∇W が有限 ε でも厳密に成立**する
   (v1.5 実装で統一、L5 解消。V9 で機械検証)。ε→0 で原仮定「決定力は m/d」に一致し、
   「引力はその微分」という原仮定と厳密に整合する。遠方漸近・d=0 の値(m/ε)は旧式と同一。
3. **スピン引きずりの減衰則** — ω_j(d) = s_j·(R_j/(R_j+d))^q、q=2 を既定とした。
   d→0 で剛体回転(表面速度 s·R)に接続し、遠方では ω·d ~ 1/d と減衰して重力(1/d²…積分的には
   ポテンシャル1/d)より短距離で効く。q はパラメータとして編集可能。
4. **エネルギー整合** — 「速度→スピン変換」は2次元円盤の接触摩擦インパルスとして実装する。
   これにより変換の**符号が自動的に正しく**なり(かすめる衝突は接触点の滑りを打ち消す向きの
   スピンを生む。ケプラー剪断下では全粒子のスピンが同符号に揃う)、T3 の整列が創発する。
   〔v1.1: 摩擦則の符号と作用線を A13/T7 に整合するよう改訂した。E9 参照〕
5. **時間の遅れの適用範囲** — 既定では (a) 各質点の固有時計の表示、(b) 光線の屈折(T5)に適用。
   物質の力学への適用(深い重力井戸内でスローモーションになる)はトグルで選択可とする
   (常時適用は数値コストと安定性の理由で既定オフ。補完としてここに明記)。
6. **保存則について〔v1.1 で改訂〕** — 旧版では「大域の運動量・角運動量は厳密には保存しない」
   としていたが、原仮定者の相互性公理 A13 により**厳密保存**へ改訂した(T7)。
   フレーム引きずり(E6′)は引きずった側への反作用を伴い、背景 D₀ との交換分は
   無限慣性リザーバへの移譲として帳簿づけされる。k_frame = 0 とおくと
   通常のニュートン力学に退化するため、比較検証が可能である点は変わらない。
7. **スピンの符号** — スピンは符号付き(反時計回り正)。「熱」としては s² を用いる。
   逆向きスピンの接触は摩擦で互いに打ち消し合い、熱としては散逸する(物理的に自然な帰結)。
8. **放射の扱い〔v1.1〕** — A12 の放射は現状スカラーエネルギーとして帳簿づけし、
   光量子の運動量(放射反作用)はモデル化しない(未解決問題 L11)。放射冷却 η_rad の
   既定値は 0(オフ)とし、既存デモの挙動を変えない(冷却実験はパラメータで有効化する)。
9. **決定力の解釈補足〔v1.7・原仮定者承認 2026-07-19。正本は BOX_UNIVERSE.md §2〕** —
   ①**慣性のアンカー**: W の強さは「静止・等速の基準(局所フレーム u)が源の運動に

#### 一様重力場 gravityX / gravityY〔v1.17〕
決定力勾配(E4)とは独立に、全自由粒子へ一様な外部加速度を加える:
v̇ᵢ += (gravityX, gravityY)。決定力 W には寄与しないため、時計(E7R)・光(E8R)・
フレーム引きずり(E6′)を一切歪めない。地上実験室(自由落下・対流・沈降)の
デモに使う。運動量・角運動量の変化は外場との交換としてリザーバ帳簿
(resP/resL)に記録され、恒等式 P + resP = 一定 は維持される(下記)。
D₀ が「遠方宇宙の背景決定力(マッハ原理的安定化)」を表すのに対し、
gravityX/Y は「局所的な外部場(地表重力など)」を表す — 役割が異なる。

v1.3 の QA で確立した本法則系の帰結(付録D 書き戻しの要約、v1.9 でも有効):
**接触した物質塊の唯一の熱的安定点は s=0**(E9 の接線摩擦が接触網の同符号スピンを
転がり整合へ散逸させる「ギア冷却」)。ゆえに (i) 伝導路は非接触の疎な配置でのみ機能し
(E10′ は距離減衰付きで遠達する)、(ii) 圧力で支えられた温かい星は接触塊ではなく、
スピンを持つ中心質量の E5′ クッション(保存力=散逸しない。V10b)に支えられた
運動する気体圏として実現される。
