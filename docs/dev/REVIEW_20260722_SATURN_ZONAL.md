## 1. 裁定の要約

**採用の柱**: ChatGPT 提案の「帯状重力調和係数 J2〜J12 による現実較正」を **E13** として
beta に実装した。E12(測地線1PN)と同じ「現実対応デモ」層であり、DFM 公理系の帰結ではない。
検証: 解析式は D68 リングレットで ϖ̇=38.231443°/日を返し、レビュー報告値と**桁まで一致**、
Cassini 観測 38.243±0.008°/日 と相対差 −0.03%(QA `zonal.analytic-d68` が固定)。

## 3. 実装成果物(beta のみ)

1. **E13 エンジン**: 全対ループ直後に扁平中心天体→他粒子の補正加速度(作用反作用対称)。
   `zonalA()` 前計算+`zonalApsidal()` 解析式(単位系任意)を `HP.zonal` に公開
2. **🧭 saturnZonal プリセット**: 中心(m=1500, pinned, zonal: R_S=100, Cassini J2〜J12, C=1)+
   テスト粒子10(a=108〜252, e=0.05, 近点+x整列, DFM効果ゼロ)。差動近点移動が肉眼で見える。
   **実装上の要点(QA実測で較正)**: 初版 QA の実測は 1.4°/周と解析値 8.8° から大きく外れたが、
   原因は物理ではなく**測定の汚染**だった — ①開始点がケプラー初期条件のため J 補正下では
   真の近点でなく、開始直後の擬似極小を近点として拾っていた(ウォームアップ k>3000+
   近点ゲート r<103 で排除)②初期条件が名目 a=108 に対し実効軌道は a_eff≈104.2 に落ち着くため、
   解析値は **a_eff=(rmin+rmax)/2** で評価する必要がある。修正後は
   **実測 9.464°/周 vs 解析 9.529°(相対差 0.68%)**で一致。physics は geoPN=1・λ_PN=0
   (☿ の λ=0 統制と同じ E6′排他+純ニュートン構成。kFrame=0 なら等価だが、スライダーで
   kFrame を上げても E6′ が復活しない防御的分離)
3. **🪐 説明文訂正**(C1/C5)+ **E13 法則解説**(LAWS ja/en)
4. **QA 4項目**: 解析 D68(38.231443±0.05 / 較正 38.243±0.002)・点質量退化・
   実測近点移動(解析比 20% 以内)・二重計上防止(J=0 で数値基線 <1.15°/周)。
   E13 の無い対象(ルート版)では自動 SKIP
5. **PHYSICS.md**: §5 に E13 小節(beta 先行の明記)・§6 表に saturnZonal 行
