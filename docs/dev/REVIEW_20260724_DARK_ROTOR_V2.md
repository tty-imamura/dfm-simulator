## 2. スピン役割分離の再考 — 温度の粗視化解釈(設計)

原仮定者裁定により温度の意味論が確定した:

> 大きな粒子は「小さな粒子の塊」の近似である。温度=構成粒子のミクロなスピン(乱雑な内部
> 回転)であり、マクロな spin はコヒーレントな回転。冷たい高速回転体が急停止すると、その
> エネルギーはミクロスピンへ移り「熱い停止した粒子」になる。ミクロスピンは光として放出され、
> 温度として認識される。

この解釈の帰結(4-43 の再定位):

1. 温度は本来 **spin と独立の内部状態変数 T_int** である。現行 A9(T=Is²)と 4-43 の
   sTherm はその暫定射影(sTherm·s = スピンのうち「ミクロ成分」とみなす割合)
2. 完全実装 = **A9 改訂スプリント(台帳 4-51・要承認)**: E9 散逸→T_int 蓄積・
   E10′=T_int の伝導・E5′ 圧力=T_int 起源・E11=T_int の放射。熱デモ群(♨️🎈📏❄️等)の
   初期条件移行と QA 再較正を伴う大規模変更のため、単独スプリントとして設計してから着手する
3. **4-50(E10′「冷たいのに熱伝導」問題)はこの裁定で解決方向が確定** — κ_s は本来
   T_int の伝導であり、実装は 4-51 に統合してクローズ

## 4. 較正実測(3ラウンド)— ダークローターの安定設計則

> 全て seed 20260723・銀河円盤380粒子。ハロー20体(m30)+中心 m600。数値は
> 外縁帯 v_φ(円盤)/ r90 / ハロー平均|spin|。

**R1: 散逸ゼロ+コア付きハロー(ChatGPT 推奨B 相当)** — 暴走。
ハロー外殻スピン 0.1→**8.1**(6000步)・r90 1428・外縁 v_φ 0.36 に崩壊(中心ローター無し変種)。
中心ローター有りでも halo|spin|→1.2〜1.5・r90 ≈316〜591 で不安定寄り

**R2: 散逸あり+コア付きハロー** — さらに暴走(μF の速度→スピン変換も coreSR に増幅される)。
halo|spin| 0.1→**23.5**・r90 2853・ハロー散逸 19/20 喪失。
**→ 設計則: 軽い天体(m30)へのコア付与は E6′ 反作用トルク×coreSR の正帰還で不安定**

**R3: 採用構成(G)= pinned 中心コアローター+コア無しハロー+散逸平衡** — 安定。

- 中心(pinned)は E9/E10′/E11/E6′ トルクの対象外 — スピン**厳密不変**・反作用はリザーバ帳簿
- u_φ(140/200/260) 全ローター on/off 比 = **3.67 / 2.54 / 1.92**(t=0 決定論・QA 閾値 >1.8)

  従い「中心は固定(外部拘束)」を説明文に明記)

  引きずりで、スピン寄与は過渡的(既知の 4-51 領域として記録)
