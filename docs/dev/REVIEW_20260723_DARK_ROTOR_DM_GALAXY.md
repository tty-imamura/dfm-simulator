## 1. 裁定の要約

1. **「高速回転で光がすり抜ける」現象は DFM 内部で再現可能な実現象**である。ただしその正体は
   「ブラックホールの透明化」ではなく、**回転する決定フレームによる光線の掃き出し**
   (媒質随伴項 k_F·u·(n_eff/c₀) が光線方向項 c₀/n_eff を上回る χ≳1 領域の帰結)。
   実測: few-shot BH 条件で捕捉は **spin 0.52→0.56 で 5/5→0/5 に急崩壊**(§3)
2. **物理対応ロック K_t=c₀²/G=3600 では spin 0〜20 の全域で光子捕捉が最初から存在しない**。
   現象は一般化トイ設定(K_t=40)に依存する(§3.4)
3. 銀河実験(§4)では: **ハローの質量効果は明瞭**(外縁 v_φ +25%)。しかし現行公理
   (spin=フレーム源=熱=斥力)のままでは、**高スピンハローは E5′ 斥力で円盤を破壊し**
   (r90 223→680)**ハロー自身も蒸発する**(20体中9体が r>400 へ散逸)。スピン斥力を
   切った統制(kRep=0)でのみ、**整列スピンの純フレーム効果 +6.7%(6000步)〜+11.6%(9000步)**
   が現れ、**ランダムスピンでは +0.5〜2.2% に消失**(Grok 予測の実測確認)
4. **総合裁定: 「高速回転ブラックホール=ダークマター」は現行 DFM 公理のままでは不成立**
   (ChatGPT 判定を実測で確認)。ただし **スピン役割分離(台帳 4-43)後の「DFM ダークローター」
   は反証可能な条件付き有望仮説**として再挑戦の枠組みを台帳 4-47 に登録
5. ロードマップ反映: **4-43 を P0 へ昇格**(本仮説検証の前提条件と実証)・4-47(再挑戦条件)・
   4-48(QA/文言の実態整合)・4-49(回転曲線基準線)新設・**4-20(多seed)優先度上昇**
   (T4 増強の seed 依存が実測された — §4.3)

### 3.2 物理対応ロック対照(K_t=c₀²/G=3600)

spin 0, 0.5, 1, 2, 5, 10, 20 の**全域で捕捉 0**(QA5本・ファン100本とも)。spin=0 でも
n_eff(0)=1.745 では光線はほぼ直進偏向で通過する(minR≈衝突径数)。すなわち
**「光子捕捉」も「高スピンでの捕捉消失」も、物理対応条件の外(K_t=40)のトイ現象**である。
なお極端スピン(s≥5)では物理ロック下でも minR が押し上げられる(s=20 で 182〜204)—
フレーム随伴そのものは K_t に依存しないため、掃き出し効果だけは残る(記録)。

1. **質量効果は成立**: halo-s0 / base = **+25%**(6000步)。通常のダークマター的な
   「見えない質量がポテンシャルを深くする」経路は DFM でもそのまま機能する
2. **現行公理の高スピンハローは銀河を壊す**: kRep=1 の全スピン変種(整列/反転/ランダム)で
   円盤 r90 が 517〜680 へ膨張(基準 183〜223)し、外縁 v_φ は base 以下へ低下。整列が最悪
   (680)。ハロー自身も自己斥力で蒸発する(整列: 12000步で 20体中 r<400 残存 11体・

   (ChatGPT §5.2 の実測確認)。kFrame=0 でも同様に壊れる(halo-aligned-kF0: r90=586)—
   破壊はフレームでなく E5′ 斥力起因
3. **純フレーム効果はスピン整列時のみ**(kRep=0 統制): 整列/spin0 比 = **+6.7%(6000步)

   フレーム速度を mass-only 比 +27% 押し上げる(6000步: 0.967 vs 0.760 @r=180)
4. **ハローは「冷たく回り続ける」ことができない**: kRep=0 ではハローが円盤内へ沈み、
   接触摩擦(μF)とスピン拡散(κ_s)で **|spin| 平均 2.0→0.88(6000步)→0.33(12000步)**
   と急減(放射 etaRad=0 でも失われる)。kRep=1 ではスピンは保つが上記2のとおり系を壊す。
   **スピン保持と系の安定が現行公理では両立しない** — 4-43(役割分離)が前提条件である
   ことの直接実証
5. **スピン0でも「共回転する質量ハロー」はフレームを引く**(新知見・記録): halo-s0 の
   u_φ(140)=1.02(base 0.45)— E3 の並進項により、軌道運動する重いハローはスピンなしでも
   決定フレームを押し上げる。DFM 宇宙で「ダークマター的増強」を担う主経路は、天体スピン
   ではなく**共回転する質量分布の並進引きずり**である
6. **T4 増強の seed 依存(重要・4-20)**: 本 seed(20260723)の base A/B 比は 6000步で
   **1.007**(QA seed の実測 1.082 に対し)、12000步で 0.875 へ反転。既知の「有限時間ゲート」

   多seed 検証(4-20)の優先度を上げる

## 5. 「銀河の構造」への DFM 的解明 — 本便時点の総括

- **回転曲線**: 外縁増強は (a) ハロー質量(+25%)、(b) 共回転質量の並進フレーム引きずり、
  (c) 整列スピンのフレーム寄与(kRep 切離しで +7〜12%)の3経路の和として分解計測できた。
  漸近平坦化には外縁で u_φ≈V_f/k_F の一定フレームが必要(ChatGPT §6.3)であり、
  q=2 の単一/少数天体では作れない — **分布ハロー+整列が唯一の候補**(4-47)
- **銀河形態**(Grok §3.2): 「整列→渦巻的・ランダム→楕円的」は現行公理では検証不能
  (ランダム変種も斥力破壊が卓越)。スピン分離後の再検証項目
- **「暗さ」**: 現行実装の暗さは ①RAY_MASS_MIN 未満の計算省略 ②etaRad=0 ③高スピンでの
  光線掃き出し(トイ設定依存)の3つの人工的/条件付き機構の混合であり、
  **「観測的に暗いが重力する天体」の物理的説明にはまだなっていない**
- **結論**: 「高速回転 BH=DM」は現行公理では不成立。ただし DFM は「質量+並進引きずり+
  整列スピン」という**通常 DM とは異なる合成増強機構**を持ち、スピン役割分離(4-43)後に
  u_req 逆問題(§2.1 表の中心式)で分布・整列を設計する **「DFM ダークローター」仮説は
  反証可能な数値仮説として有望**(4-47)

## 6. 反証条件の判定(ChatGPT §13.3 の9条件)
