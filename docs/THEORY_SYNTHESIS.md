## 1. 名称と一文定義(確定)

- **和名**: 決定力場モデル
- **英名**: **Determinacy-Field Mechanics (DFM)**(「決定力」= *determinacy*、場 W = *determinacy field*)
- **一文定義**: 空間座標の「決まりやすさ」を表す単一スカラー場 W(決定力)から、
  慣性の基準・重力・時間の進み・光の屈折を導き、質点スピンを熱と同一視することで
  熱力学的挙動までを創発させる、関係論的(マッハ的)力学のトイモデル。

命名理由: w = m/√(d²+ε²)〔v1.5 で Plummer 型に統一。ε→0 で m/d〕は「その点の座標を
どれだけ強く確定させるか」の重みであり、
determination strength の直訳より *determinacy*(確定性)が学術英語として自然。
略称 DFM は物理学で既占有の主要略称と衝突しない(執筆時に再確認を指示済み)。
不採用案: "Machian Weight Field"(Mach の名を冠すると Mach 原理の特定定式化だと
誤解される)、"Relational W-field model"(内容を示さない)。

## 3. 中核的主張(Contributions として論文に列挙する6点)

- **C1(最小性と実装可能性)**: 公理10個・式10本の体系が内部矛盾なく閉じ、

- **C3(弱場GRとの構造対応)**: パラメータ写像 K_t = c²/G、k_L = 2G/c² の下で、
  弱場一般相対論の時間遅れ・光線偏向と同型の式構造を持つ(§4)。

  ニュートンのバケツ問題を「実験可能なパラメータスキャン」に変換する装置である。

  円盤銀河系で回転曲線が平坦化する(k_F=0 ではケプラー減衰)。

- **C6(スピン=熱公理からの熱力学の創発)**: 接触摩擦による並進→スピン変換、

- **C7(厳密な保存則)〔v1.1〕**: 相互性公理 A13 と改訂式 E5′/E6′/E10′ により、

## 4. 対応原理(Correspondence)— 論文 Sec. IV の確定内容

### 4.1 ニュートン力学への退化
k_F = 0(フレーム引きずり無効)かつスピン諸項無効のとき、E4 のみが残り
標準的なニュートン重力 N 体系に厳密に一致する。**モデルは比較対象を内蔵している**
(同一初期条件で k_F だけ変えた対照実験が可能 — T4 のグラフがまさにこれ)。

### 4.2 弱場一般相対論との写像(導出済み・執筆者は転記のみ)
Φ_N = −G·D(ニュートンポテンシャル、D = Σ m_j/d_j、ε→0)とおく。

| 量 | 弱場GR | DFM | 一致条件 |
|----|--------|-----|----------|
| 固有時 | dτ/dt ≈ 1 + Φ_N/c² = 1 − G·D/c² | dτ/dt = 1/(1+W/K_t) ≈ 1 − (D₀+D)/K_t | **K_t = c²/G**(D₀項は時計の大域再校正) |
| 屈折率 | n ≈ 1 − 2Φ_N/c² = 1 + 2G·D/c² | n = 1 + k_L·W | **k_L = 2G/c²**(D₀項は定数屈折率=偏向に無寄与) |
| 重力 | a = −∇Φ_N | a = G·∇D(E4) | 恒等(定義より) |
| フレーム引きずり | Lense–Thirring 歳差 Δϖ_LT ∝ G·J/(c²√(GM)·r^{3/2})(J = I·s) | E6′ による近点移動 Δϖ ∝ k_F·S_c·R_c²/√(GM·r³)(DERIVATIONS §5a) | **同一の r^{−3/2}・スピン×半径² スケーリング**。対応 k_F ↔ 2G·m_c/c²(規格化)。強度 k_F は意図的増幅であり量的一致は主張しない。遠方(D₀r ≫ M)では u_c ∝ 1/r² に折れ、背景が歳差を遮蔽する(DFM 独自予言)〔v1.1 で確定〕 |

**構造的相違(論文で明示する誠実ポイント)**: GRでは時間遅れと光偏向が単一の計量から
従い比率が固定されるが、DFMでは K_t と k_L が独立。すなわち DFM は拘束の弱い
「スカラー光学的重力」であり、GR の代替ではなく**教育・探索用の実験場**である。
また光偏向の因子2は k_L の校正で再現するのであって導出ではない(脚注で明記)。

### 4.3 次元復元
シミュレータは無次元(G=1)。論文では理想式(ε→0)を SI に復元する表を付録に置く:
[W] = kg/m、K_t = c²/G ≈ 1.35×10²⁷ kg/m、k_L = 2G/c² ≈ 1.48×10⁻²⁷ m/kg。
ε は数値正則化であり理論の一部ではない(§7 L5)。

## 6. 主張しないこと(Negative claims — 論文の防御線)

論文の信頼性は**過剰主張の排除**で決まる。以下を Introduction と Discussion の両方に明記する:

1. DFM は実宇宙の記述だと主張しない。「もし空間が純関係的で引きずりが強かったら」という
   **反実仮想の公理系の帰結を調べる理論実験**である。
2. T4 は「ダークマター不要」を意味しない。実宇宙の GR ではフレーム引きずりは
   v/c² 抑制で桁違いに弱く、DM の観測証拠(CMB・重力レンズ統計・構造形成)は独立に強い。
3. ローレンツ不変性・因果的伝播はない(u 場は瞬時に決まる遠隔作用。§7 L8)。
4. 2次元モデルであり、3D 定量予言は行わない。

## 7. 既知の限界と未解決問題(論文 Discussion / Future Work の確定リスト)

- **L1 作用原理の欠如**: E6 は運動方程式レベルの規則(Du/Dt の一次近似)であり、
  ラグランジアンから導かれていない。変分定式化の探索が第一の未解決問題。
- **L2 大域保存則の破れ → 〔v1.1 で解消〕**: 原仮定者の相互性公理 A13 と
  改訂式 E5′/E6′/E10′(DERIVATIONS §0〜§1)により、線運動量・全角運動量は厳密に保存(T7)。
  D₀ との交換は無限慣性リザーバへの移譲として帳簿づけされる(リンゴと地球)。
  論文では旧版の非保存形式を脚注で言及し、保存形式を正とする。
  **〔v1.6 表現明確化〕** D₀=0・pinned なし・境界なしの場合が**閉鎖系の保存則**であり、
  D₀>0 または pinned ありは**外部環境を含む開放系の収支**である。resP/resL は
  「保存量」ではなく「境界との交換量」と表記する(DERIVATIONS §10.1 裁定 #8)。
- **L3 エネルギー収支の部分的閉性 → 〔v1.1 で解消〕**: 放射公理 A12(ウィーン型
  スペクトル+SB型光度)を導入し、γ_n 散逸を衝突閃光の放射として計上。
  並進→スピン(摩擦)/放射(法線減衰)、スピン→放射(冷却)で第一法則が閉じる。
  A12 は設計者補完であり、**2026-07-15 に原仮定者が承認済み**(正式に公理系の一部)。
  **〔v1.5 追補〕v1.1 の閉性は不完全だった** — 接線摩擦のすべり散逸と E10′ 同期の
  回転エネルギー損が帳簿外(等質量2円盤の全付着で ΔE=−1/12 が消滅する反例)。
  v1.5 で A12′ として radE に計上し**完全閉性**を達成(V10 で機械検証。DERIVATIONS §9)。
- **L4 状態方程式が未導出**: スピン斥力ガス(A10)の EOS は理想気体 PV=NkT 型か不明。
  斥力が ω²d の対相互作用である以上、ソフトコア流体に近いはず。数値測定は好個の続報テーマ。
- **L5 正則化の二重化 → 〔v1.5 で解消: E1 を Plummer 型へ統一〕**: E1′ = m/√(d²+ε²) により
  ∇w が E4 の (d²+ε²)^{−3/2} 型と厳密に一致し、A6(a = G∇W)が有限 ε でも成立する
  (V9 で機械検証)。n・∇n・W・E4 の4者が単一の場に統一された(DERIVATIONS §9)。
- **L6 スピン拡散の到達距離**: A9 の文言は「近接した質点」だが E10 のカーネル g(d) は
  滑らかな短距離関数で接触外でも作用する。論文では「short-range kernel」と正確に記述する。
- **L7 3D 拡張**: スピンのベクトル化、A8 引きずり場のベクトル場化、回転面整列 T3 の
  3D 版(角運動量整列)が自然な拡張。
- **L8 因果性**: u と W は各時刻の全質点配置から瞬時に決まる(ニュートン重力と同種の
  遠隔作用)。有限伝播速度(遅延ポテンシャル化)は未実装・未定式化。
- **L9 異粒径系の温度平衡〔v1.1 新規〕**: E10′ の平衡点は「s の均一化」であり、
  粒径が異なると T = I·s² は完全一致しない(s 均一と T 均一の不一致)。厳密化は未解決。
- **L10 渦巻密度波の分散関係〔v1.1 新規〕**: (G5) の腕の巻き込みは幾何論に留まる。
  DFM 回転則(G1)上の密度波の完全な分散関係の導出は未解決。
- **L11 放射反作用〔v1.1 新規〕**: A12 の放射は現状スカラーエネルギーとして扱い、
  光量子の運動量が持ち去る反作用と T7 との整合は未定式化(放射が持ち去る角運動量は
  帳簿変数 radL として計上するに留める)。
- **L12 物質側ローレンツ応答の欠如〔v1.6 新規〕**: E8′ は光を媒質(決定フレーム)に
  随伴させるが、移動する物差し・共振器・原子遷移の応答は未定義であり、
  Michelson–Morley / Kennedy–Thorndike 型実験の等方性は再現されない。
  §6 の Negative claims と README「位置づけと限界」の理論的正体。教育用・解釈モデル
  (方針A)では仕様であり、検証可能な優先フレーム理論(方針B)に進む場合の最重要課題
  (DERIVATIONS §10.7)。
- **L13 K_t と k_lens の独立性〔v1.6 新規〕**: 時計率と屈折率が独立係数を持ち、
  「単一場が時間と光を導く」という中核思想に自由度が余る。単一ポテンシャル
  ψ = W_ext/K_t からの統一(N=e^{−ψ}、A=e^{+ψ}、n_eff=A/N。E7R/E8R)で解消可能。
  確定設計は DERIVATIONS §10.4、実装は v1.7 候補(原仮定者承認待ち)。
