## 0. 公理の改訂と追加

### A13(相互性)— 原仮定者による改訂 ★最重要
> 空間の引きずりは、厳密には反作用を生じ、運動量を保存する。
> リンゴが地球に落ちるとき、地球もリンゴに引っ張られている。

**公理文**: すべての相互作用(重力・スピン斥力・フレーム引きずり・接触・スピン拡散)は
対ごとに作用反作用対称であり、系全体の線運動量と全角運動量
L_tot = Σ m_i(**r**_i×**v**_i) + Σ I_i s_i を厳密に保存する。
背景決定力 D₀ との運動量交換は、静止した無限慣性リザーバ(遠方宇宙)への移譲として
帳簿づけされる — リンゴに引かれた地球が、感知できないほど僅かに動くのと同じである。

これにより旧 THEORY_SYNTHESIS §7 の限界 **L2(保存則の破れ)は解消**され、
「保存則を厳密に満たす関係論的力学」として理論の格が一段上がる(新定理 T7)。
実装上の改訂は §1 の E5′/E6′/E10′。

### A11(光の伝播と同時性)— 原仮定者による改訂
> 光は「決定した空間」を進み、観測者によるパラドックスは発生しない。

**公理文**: 光は各点の局所決定フレーム **u**(**r**) に対して等方的に速さ c₀/n(**r**) で
伝播する。座標時間 t は宇宙で一意(絶対的同時性)であり、固有時間 τ の遅れ(E7)は
時計の**動的な**遅れであって、運動学的な相対性ではない。
したがって同時性の相対性に由来するパラドックス(双子・同時刻の不一致)は原理的に生じない。
DFM は「優先フレームを持つ理論」であることを明示する — 決定力場そのものが媒質である。
帰結: 光は媒質 **u** に運ばれる(フレネル随伴の完全版、E8′)。回転体の左右で
光の湾曲が非対称になる(新帰結 T8 — Gemini が独立に提案したブラックホール実験と一致)。

### A12(放射)— 設計者による補完(**原仮定者承認済み 2026-07-15**)
D1(§2)の導出根拠と、旧限界 L3(法線減衰 γ_n の散逸先が不明)の解消のため追加する。

**公理文**: 温度 T_i を持つ質点は光量子を放射する。スペクトルのピーク波長は
ウィーン型則 λ_peak = b_m / T_i に従い(b_m はモデル定数)、光度は
Λ_i = η_rad · T_i^p(既定 p=4、シュテファン=ボルツマン型)。
E9 の法線減衰 γ_n で失われる並進エネルギーは、衝突時の閃光として放射される。
これによりエネルギー収支が閉じる: 並進 → スピン(摩擦)or 放射(法線減衰)、スピン → 放射(冷却)。

## 2. D1: 温度と光の波長からスピン速度の方程式を導く

A12(ウィーン型則)と温度定義 T = I s²/k_B を連立する。

> **s_i(λ) = √( k_B b_m / (I_i λ_peak) ) = (1/R_i)·√( 2 k_B b_m / (m_i λ_peak) )**

表面(接線)スピン速度はさらに簡潔になる:

> **v_surf = s_i R_i = √( 2 k_B b_m / (m_i λ_peak) )** — 半径に依らず、質量と観測波長だけで決まる

**帰結**:
1. 青い(λ 小)天体ほど速く自転する: s ∝ λ^{−1/2}。
2. 同じ色の2天体では、重い方がゆっくり自転する: s ∝ 1/(R√m) ∝ m^{−1}(R=r_R√m のとき)。

4. 気体デモの平衡(E10′: s 均一)では、大粒子ほど T = Is² が高く赤外側から可視側へずれる —
   実装の色マップ改修(§0)と整合する検証項目になる。

シミュレータ単位系(k_B=1): λ = b_m/(I s²)、s = √(b_m/(I λ))。b_m は UI パラメータ化する。

## 3. D2: 楕円銀河・レンズ状銀河・渦巻銀河の形状方程式

### (G1) 回転則 — DFM の銀河基本式

u_φ(r) 場の中を周回する星に **向心加速度 k_F u_φ v_φ/r** を与える(導出: 経路に沿う

> v_φ²/r = g_eff(r) + k_F u_φ(r) v_φ/r、 g_eff = G ∂_r D − a_rep,rad

これを v_φ について解くと **v_φ(r) = ½[ k_F u_φ + √( k_F² u_φ² + 4 r g_eff ) ]**。

### (G2) フレーム場の自己無撞着方程式と平坦域の長さ
u_φ 自身は内側の星々の回転から作られる(不動点方程式):

> u_φ(r) = [ ∫ Σ(r′) w(|**r**−**r**′|) v_φ(r′) dA′ + Σ_c w_c ω_c r ] / W(r)

円盤質量 M_d、平均回転速度 V̄ に対し外縁では w 総和 ≈ M_d/r なので
u_φ(r) ≈ V̄ · M_d/(M_d + D₀ r)。したがって:

> **平坦域**: r ≪ r_flat では v_φ ≈ k_F V̄(平坦)。 **r_flat ≈ M_d / D₀**
> r ≫ r_flat では u_φ → 0 となり v_φ → √(GM_d/r)(ケプラー減衰に復帰)

平坦部の自己無撞着条件 v_φ ≈ k_F v_φ + (重力項) は **k_F = 1 が特異的に自己整合的**で
あることを示す(k_F < 1 では平坦レベルが内側平均より下がる)。
**モデル内予言**: 回転曲線は無限に平坦ではなく、半径 M_d/D₀ で折れる。D₀ を上げると
平坦域が縮む — シミュレータで直接検証可能。

### (G3) 形態分類 — 3つの無次元数

  エピサイクル振動数 κ_eff² = r dΩ_eff²/dr + 4Ω_eff²、Ω_eff = (v_φ − k_F u_φ)/r)

**分類則**(境界値 h_c, Q_c は数値校正、初期見積り h_c ≈ 0.3, Q_c ≈ 1.5):

## 4. D3: 土星の環を安定形成する方程式

### (R1) 軌道則と回転方向の選択
中心天体(質量 M、スピン S_c、半径 R_c、静止)まわりのリング粒子の円軌道は (G1) の特別な場合:

> v_φ(r) = ½[ k_F u_c(r) + √( k_F² u_c(r)² + 4GM/r ) ]、
> u_c(r) = [ w_c/W ]·ω_c(r)·r = [ M/(M + D₀r) ]·S_c (R_c/(R_c+r))^q · r

u_c > 0(順行)側は必要な |v − u| が小さく、逆行粒子は大きなフレーム逆風を受けて
衝突散逸(E9)が激しい。ゆえに **環は中心天体のスピンと同方向に淘汰される**
(実在の土星の環が自転と同方向である事実の、モデル内での説明)。

### (R2) DFM ロッシュ条件(引きずりせん断つき)
粒塊(質量 m、密度 ρ)が潮汐+フレームせん断で引き裂かれない条件。
引きずりせん断数を **𝒟(r) ≡ k_F · | r ∂_r( u_c/r ) | / Ω_K(r)**、Ω_K = √(GM/r³) と定義して:

> **ヒル半径(2D)**: r_H = r · √( m / (3M) ) · [ 1 + c_R 𝒟(r) ]^{−1/2}
> 粒塊サイズ δ < r_H なら生存(衛星化)、δ > r_H なら破砕(環に留まる)
> 等価な臨界密度形: ρ_crit(r) = ρ_Roche,Newton(r) · [ 1 + c_R 𝒟(r) ]

(c_R は O(1) の校正係数。3D 移植時は指数 1/2 → 1/3。)
**帰結**: フレームせん断は実効ロッシュ限界を**外へ押し広げる** — DFM の環は
ニュートン力学より広い半径帯で安定に存在できる。

### (R3) 熱平衡幅と安定窓
衝突加熱と放射冷却(A12)の釣り合いが速度分散 σ_eq を決める:

> μ_f · n(r) σ_coll ⟨Δv²⟩ · μ̄ / 2 = η_rad Σ_i T_i⁴ / N ⇒ σ_eq(r)
> 環の厚み(動径幅): w_ring = σ_eq / Ω_K

**環の安定窓**(3条件の同時成立):
1. 重力凝集しない: Q_ring = σ_eq Ω_K/(πGΣ_ring) > 1 ⇔ σ_eq > πGΣ_ring/Ω_K
2. 衛星化しない: ρ_ring < ρ_crit(r) (R2)
3. 粘性拡散が遅い: t_spread = (Δr)²·Ω_K/σ_eq² ≫ 観測時間

1〜3 を同時に満たす半径帯 [r_in, r_out] が「安定な環」。σ_eq は自己調整的
(σ が上がる→衝突減→加熱減→σ 低下)なので、窓の中では **環は熱的アトラクタ**である。

### 10.2 アナロジーの定理化 — T9(時間パラドックスの座標化)

現行の基礎表示では時間(E7: K_t)と光(E8′: k_lens)が独立係数を持つ。観測者を物質共動
