# 仮想物理ラボ — 仮定物理法則シミュレータ

「空間座標は質点の相対位置で決まる」「スピン=熱」といった**仮定の物理法則**を体系化し、
その帰結(回転方向の整列、平坦な銀河回転曲線、熱平衡、光の湾曲、空間の引きずり)を
iPhoneのブラウザで観察できるシミュレータです。

## 使い方(iPhone)

`index.html` を開くだけで動きます。ホスティング不要の方法:

1. **GitHub Pages(推奨)** — リポジトリの Settings → Pages → Branch を `main`(root)にして保存。
   数分後に `https://<ユーザー名>.github.io/HypotheticalPhysics/` をSafariで開く。
   共有→「ホーム画面に追加」でアプリのように使えます。
2. **ファイルとして開く** — `index.html` をiCloud Drive等に保存し、
   「ファイル」アプリからSafariで開くことも可能です。

## 仮定物理法則(要約)

1. 空間は実体を持たず、質点同士の相対配置だけで決まる(絶対空間なし)
2. 空間座標の「決定力」= 質量÷距離。遠方の大規模構造は背景決定力 D₀ として効く
3. 決定力の勾配が重力(ニュートンの逆二乗則を自動再現)
4. 質点は「近くの空間」に対して慣性を持ち、空間ごと引きずられる(フレームドラッグ)
5. 決定力が高い場所の時計は遅く、空間(共動フレーム)に対して速く動く時計も遅れる。
   光は同じ ψ=W/Kt で屈折して曲がる(v1.7 で時間と光を単一の ψ に統一。n=e^{2ψ})
6. スピン=熱(温度 T=Is²)。衝突で速度がスピンに変換され、スピンは斥力(圧力)を生み、拡散して平衡に達する
7. すべての相互作用は作用反作用対称で、運動量・角運動量を厳密に保存する(遠方宇宙との交換は帳簿づけ)。温度 T の質点はウィーン則 λ=b_m/T の光を放射する

公理系(A1〜A13)・帰結(T1〜T8)・計算式(E1〜E11)の全文は
[docs/PHYSICS.md](docs/PHYSICS.md) を、公理改訂の導出は
[docs/DERIVATIONS.md](docs/DERIVATIONS.md) を参照してください。

## 本プロジェクトの位置づけと限界

本シミュレータは「仮定の物理法則」(決定力場モデル: DFM)の帰結を観察する思考実験・教育用ツールであり、現実の物理学の代替理論を主張しません。

- A11 は優先フレームと絶対的同時性を仮定します。v1.7 の ψ 統一により、**時計と光の弱場一次効果** — GPS 時計の進み(+38.5 μs/日)・太陽縁の光偏向(1.7512″)・シャピロ遅延 — は写像 Kt=c²/G ひとつで定量再現します(第3次レビューで検算・機械検証。DERIVATIONS §12)。一方、**物質軌道の 1PN**(水星近日点移動 43″/世紀 — E6′ の引きずりは GR のレンズ=サーリングと同符号の微小な「後退」のみを与える)・連星パルサーの重力波減衰(瞬時遠隔作用 L8)・物質側のローレンツ応答(Michelson–Morley 型の光速等方性 L12)は再現せず、目的外です(THEORY_SYNTHESIS §6 Negative claims)。
- 「スピン=熱」(A9)とスピン斥力による圧力(A10)は、実在の分子運動論・等分配則とは異なる仮定上の機構です。個別質点のウィーン型放射(A12)も黒体放射(プランク分布)の簡略アナロジーです。熱部分の正確な位置づけは「角運動量を持つ2D散逸粒子系の熱力学類似モデル」であり、理想気体の状態方程式(スピン圧は P∝ρ²T)・潜熱を持つ相転移・対流の定量則は再現しません(第4次レビュー裁定。THEORY_SYNTHESIS Negative claims 6・L15〜L17)。
- その代わり本モデルは、運動量・角運動量の厳密保存(V1)とエネルギー帳簿の閉性(V10: kRep=0 域 / V10b: 固定スピン+kRep>0 域。スピンが変わる過程での斥力ポテンシャル変化は帳簿外=L17 として明示)を機械検証で保証します。「もし空間がこのように振る舞ったら何が起きるか」を、対比材料として安心して探索してください。

## 開発・QA(Phase 1 再現性基盤)

新規cloneから1コマンドで全機械QA(42項目)を実行できます:

```bash
npm install && npx playwright install chromium   # 初回のみ
npm test          # 全42項目(約4分)。結果は tests/out/qa-results.json
npm run test:fast # 長時間挙動テストを省略した高速版
```

GitHub Actions([.github/workflows/ci.yml](.github/workflows/ci.yml))が push/PR ごとに
同じスイートを実行し、機械可読な結果JSONを artifact として保存します。
